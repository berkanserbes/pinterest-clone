# Pinterest Clone

## Clone the project

Clone the project on your local

```bash
  git clone https://github.com/berkanserbes/pinterest-clone.git
```

Go to project directory

```bash
  cd pinterest-clone
```

Install required packages

```bash
  npm install
```

Run server

```bash
  npm run start
```
